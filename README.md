# Проект 4: Место

# О чем он
Этот проект - аналог инстаграм с возможностью заводить профиль, выкладывать фото и ставить лайки.

# Технологии
 - Верстка flex, grid
 - Адаптивная верстка
 - Базовый JavaScript
